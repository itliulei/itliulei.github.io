浏览器执行 js 简介

  1. 渲染引擎：用于解析 html 和 css
  2. js 引擎：js 解释器执行代码时逐行解释每一句源码

js 组成

  1. ECMAScript：js 语法
  2. DOM：页面文档对象模型
  3. BOM：浏览器对象模型

js 三种书写方式

  1. 行内式
  2. 内嵌式
  3. 外链式

变量：本质上来说就是内存中申请一块用来存放数据的空间

变量的使用（变量初始化）

  1. 声明变量：var,计算机会自动分配内存空间
  2. 赋值：将值保存进去

声明变量的特殊情况

  1. 只声明不赋值结果是 undefined
  2. 不声明 不赋值 会报错
  3. 不声明直接赋值使用(可以使用，但不建议)

js 数据类型

  1. js 是一种弱类型或者说动态语言
  2. 相同的变量可作用不同的类型
